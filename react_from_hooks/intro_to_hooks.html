Introduction to React hooks

Function components can hook into component lifecycle and component state.

Lifecycle hooks include a component: mounting, updating and unmounting.

State can record the changing condition of a component, eg counting button clicks.

Hooks must be declared and assigned at the beginning of a function component ie outside any loops, conditions, or nested functions.

Only call Hooks from a function component.

Hooks can be used many times in a single component.
