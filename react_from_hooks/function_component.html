Function component

Function component accept props as an argument and returns JSX code.

Function component names should start with a capital letter, eg MyComponent. 

    
const MyComponent = (props) => {
  ...
  return <div />
}
    
  
    
function MyComponent(props) {
  ...
  return <div />
}
    
  
    
// Code example

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>
}
