State hook

Named after state in computer science. State records changes in a component, eg counting button clicks.

A single component can call state hooks many times.

Function return

The state hook function returns a pair of values in an array, like [x,y].

Hence we assign the hook function to a pair of variables in an array.

Eg

const [count, setCount] = useState(0); 

    
// Code example

import React, { useState } from 'react'

function Example() {

  const [count, setCount] = useState(0)

  return (
    <div>
      //... component html ...
    </div>
  )
}
