State hook

Named after state in computer science. State records changes in a component, eg counting button clicks.

First import useState from React with the following line of code:

import React, { useState } from 'react'

"useState" accepts the initial state value as an argument.

Eg "0" or "Bilbo" in the two separate lines of code below: 

      
const [count, setCount] = useState(0)

const [name, setName] = useState('Bilbo')

Function return: the state hook function returns a pair of values in an array, like [x,y]. Hence we assign the hook function to a pair of variables in an array.

To store multiple data objects, a single component may call state hooks many times.

Reading state

State information can be simply referenced by name inside curly brackets, eg "{count}" in the example below. 

      
function MyComponent() {

  const [count, setCount] = useState(0)

  return (
    <div>
      <p>You clicked {count} times</p>
    </div>
  )
}

Setting state

To set a state varibale to a new value, call the setter function you assigned with useState and pass in the new state value as an argument.

Eg

      
<button onClick={() => setCount(count + 1)}>
  Click me
</button>

Or 

      
const [name, setName] = useState('Bilbo')

function handleNameChange(e) {
  setName(e.target.value)
}

return (
  <input
    value={name}
    onChange={handleNameChange}
  />
)
