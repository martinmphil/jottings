# encounter_map

Preparatory code for computer assisted tabletop [RPG](https://en.wikipedia.org/wiki/Tabletop_role-playing_game) encounter mapping

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
