Vue basics

The following code introduces basic Vue concepts in a way consistent with the Vue3 functional approach (cf Vue2 Intro).

This page concludes with Vue components which forms the basis for the Vue's function based API.

Access to the Vue3 function-based Composition API is easily achieved with the following script tags which link to 'unpkg' Content Delivery Network (CDN).

<script src="https://unpkg.com/vue@2.6.10/dist/vue.js"></script>
<script src="https://unpkg.com/vue-function-api/dist/vue-function-api.umd.js ">
</script>

Linking to this CDN provides access to Vue's function based interface through a window object called vueFunctionApi.

Setting const veuf = window.vueFunctionApi connects your code to the “vueFunctionApi” via the prefix “veuf”. To use this API, explicitly add Vue.use(veuf.plugin) to your app.

Basic reactivity

Please feel free to visit this basic reactivity example in a single html file and then view source to see the script tags containing its Vue code.

Using veuf.value() creates a reactive property. The arguments supplied to the value function sets an initial value. For example, msg = veuf.value('Hi') sets msg to an initial value of 'Hi', and the .value part makes msg reactive.

HTML

<div id = " app">

   <div> {{msg}} </div>

</div >

JS

const veuf = window.vueFunctionApi

Vue.use(veuf.plugin)

 

var app = new Vue({

  el: '#app',

  setup() {

    let msg = veuf.value('Hi')

    return{msg}

  }

})

Binding

Please feel free to visit this v-bind example in a single html file and then view source to see the script tags containing its Vue code.

The v-bind directive binds HTML attribute to JS variables. In the following extract v-bind binds the msg reactive value to the  title HTML attribute which appears when a mouse hovers over the associated element.

HTML

<div id = "app" v-bind:title = "msg">

   Hover mouse here to see dynamic binding.

</div>

JS

const veuf = window.vueFunctionApi

Vue.use(veuf.plugin)

 

var app = new Vue({

  el: '#app',

  setup() {

    const msg = veuf.value('Page loaded on ' + new Date().toLocaleString())

    return{msg}

  }

})

Interactivity

Please feel free to visit this basic interactivity example in a single html file and then view source to see the script tags containing its Vue code.

Changing reactive values requires the .value suffix.

HTML

<div id="app">

  <p>{{msg}}</p>

  <button v-on:click="shout">Shout</button>

</div>

JS

const veuf = window.vueFunctionApi

Vue.use(veuf.plugin)

 

var app = new Vue({

  el: '#app',

  setup() {

    let msg = veuf.value('Sotto voce')

    const shout = () => msg.value = msg.value.toUpperCase()

    return{

      msg,

      shout,

    }

  }

})

Both the keywords “var” and “const” can also declare variables, but “let” seems more aesthetically appealing because pressing buttons changes msg.

Basic component

HTML

Please feel free to visit this basic component example in a single html file and then view source to see the script tags containing its Vue code.

<div id="app">

  <ol>

    <todo-item

      v-for="item in missionList"

      v-bind:mission="item"

      v-bind:key="item.id"

    ></todo-item>

  </ol>

</div>

JS

const veuf = window.vueFunctionApi

Vue.use(veuf.plugin)

 

Vue.component('todo-item', {

  props: ['mission'],

  template: '<li>{{mission.text}}</li>'

})

 

var app = new Vue({

  el: '#app',

  setup() {

    const missionList = [

      { id: 1, text: 'Write plain HTML fallback page' },

      { id: 2, text: 'Create unit tests' }

    ]

    return {missionList}

  }

})
