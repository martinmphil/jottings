Props

Props is short for "properties".

Components can store their own modifiable data as state rather than props.

Parent components control prop values and pass them down to their children. Hooks maintain this top down data flow.

Props input information into a react component from a parent via custom html attributes, eg

<CastComponent character="Gandalf" />

or

function MyComp(props) {
  return (
   <h1>To the {props.direction}</h1>
)}

ReactDOM.render(
  <MyComp direction="top"/>,
  ...querySelector('#a')
)

ReactDOM.render(
  <MyComp direction="bottom"/>,
  ...querySelector('#b')
)
