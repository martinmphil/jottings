React basics

Components

Components usually form reusable bite-sized pieces of code that return html (which subsequently renders on a page).

Components represent building blocks for constructing user interfaces.

Render function

React renders components in a webpage by calling the ReactDOM.render() function,
  (where DOM stands for Document Object Model).

For example:

ReactDOM.render(<ComponentToRender />, targetNode)

Or

ReactDOM.render(jsxToRender, targetNode)

JSX

The rendered component can be described in JSX code.
  (where JSX stands for JavaScript XML.

JSX code normally nests returned html code inside a single html element, eg <div>...</div>

Curly braces {} can execute JavaScript expressions within JSX code.

Since JavaScript reserves the word "class", JSX code defines an HTML element's class with the className attribute, eg className='myClassName'.

Similarly, use camelCase for attributes like onChange and onClick.
