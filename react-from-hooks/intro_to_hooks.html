Introduction to React hooks

Prerequisites: HTML and JavaScript

Set up your development environment as described in the React docs.

React hooks allow function components to "hook" into state and effects (and other custom-made functions).

React hooks work from within function components.

Only call hooks inside a React function component.

Hooks can be used many times in a single component.

Function components can hook into component lifecycle and component state.

Lifecycle hooks include a component: mounting, updating and unmounting.

State can record the changing condition of a component, eg counting button clicks or user input.

React tracks the order in which hooks are called. Hooks must be assigned at the top level of a function component, ie outside any loops, conditions, or nested functions. For best practice assign hooks at the beginning of function components.

   const [count, setCount] = useState(0)
